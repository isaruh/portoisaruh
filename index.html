About Me

Saya bernama Isaruh Muhammad Baga sering dipanggil Isaruh atau Isar atau terserah kalian mau panggil saya seperti apa. Alasan saya masuk RPL atau PPLG ini karena keinginan saya. Saya pikir saya terlihat aneh, terlalu banyak bicara, dan mungkin terlihat terlalu bersemangat.

Terkadang saya berpikir bahwa saya aneh karena terlalu ceria, jika memang begitu tolong abaikan saja. Saya terkadang tidak tau harus bereaksi apa selain tertawa karena saya dikenal karena tawa saya.
